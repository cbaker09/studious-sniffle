# Jupyter Notebooks

This directory contains Jupyter notebooks for exploratory data analysis and experimentation.

## Recommended Notebooks

1. **01_eda.ipynb** - Exploratory Data Analysis
   - Data overview and statistics
   - Distribution analysis
   - Correlation analysis
   - Default rate analysis

2. **02_feature_engineering.ipynb** - Feature Engineering Exploration
   - Feature creation and testing
   - Feature importance analysis
   - Feature validation

3. **03_model_training.ipynb** - Model Training Experiments
   - Hyperparameter tuning
   - Model comparison
   - Cross-validation

4. **04_model_evaluation.ipynb** - Model Evaluation
   - Detailed performance metrics
   - ROC curves and confusion matrices
   - Feature importance visualization
   - Business impact analysis

## Getting Started

```bash
# Make sure you're in the project root
cd /path/to/loan_default_prediction

# Activate virtual environment
source venv/bin/activate

# Start Jupyter
jupyter notebook
```

Then navigate to this directory in the Jupyter interface.
